# Document Similarity & Clustering Dashboard

A Streamlit application for analyzing document similarity and clustering using NLP techniques.

## Features
- Topic modeling using LDA
- Document similarity computation
- Clustering algorithms (KMeans, DBSCAN, Hierarchical)
- Interactive visualizations
- Session management and saving

## Requirements
```
pip install -r requirements.txt
```

## Usage
1. Install requirements
2. Run the app:
```
streamlit run app.py
```
3. Upload documents and configure analysis parameters
4. View results and visualizations

## File Structure
- `app.py`: Main application code
- `main.py`: Simple test script
- `requirements.txt`: Python dependencies
